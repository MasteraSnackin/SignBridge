## üéØ Recording Tips

### Before Recording

1. **Prepare Device**
   - Charge phone to 100%
   - Clear notifications
   - Set Do Not Disturb mode
   - Clean camera lens
   - Good lighting setup

2. **Test Everything**
   - Test sign recognition
   - Test speech recognition
   - Test animations
   - Test hybrid mode
   - Practice gestures

3. **Setup Recording**
   - Use screen recording app (AZ Screen Recorder, etc.)
   - Enable audio recording
   - Test audio quality
   - Set resolution to 1080p

### During Recording

1. **Speak Clearly**
   - Moderate pace
   - Clear pronunciation
   - Enthusiastic tone
   - Pause between sections

2. **Show Features Clearly**
   - Hold phone steady
   - Keep UI visible
   - Highlight important elements
   - Show confidence scores

3. **Demonstrate Real-Time**
   - Show actual latency
   - Don't speed up footage
   - Show smooth operation
   - Highlight performance

### After Recording

1. **Edit Video**
   - Trim unnecessary parts
   - Add title screen
   - Add text overlays for key points
   - Add background music (optional)
   - Add captions (recommended)

2. **Export Settings**
   - Format: MP4
   - Resolution: 1080p
   - Frame rate: 30fps
   - Bitrate: High quality

3. **Upload**
   - YouTube (unlisted or public)
   - Vimeo
   - Google Drive (shareable link)
   - Include in hackathon submission
